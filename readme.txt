Изменения:
1. Убирает ограничение на один выбранный сайд вкест.
2. Добавляет тайм аут к вращению оружия, после которого оно не будет вращаться вместе с камерой.
3. Добавляет возможность использовать товары в качестве ремонта или топлива.
4. Исправляет отображение текста в режиме ai_location_debug.
5. Добавляет световые эффекты для оружия (если поддерживается).
6. Добавляет возможно указать DamageType для BlastWave.
7. Добавляет возможность использовать IM_DEBUG импульсы для вызова LUA функций.
8. Останавливает вращение кардана при остановке машины.
9. Расширяет хранилище динамических объектов ObjContainer с 16384 до 32768 объектов.
10. Поддержка широкоформатных мониторов (за счёт "отключения" куллинга)

Установка:
1. Переместить dinput8.dll и scripts в папку с игрой (SteamLibrary\steamapps\common\Hard Truck Apocalypse).
2. По желанию изменить настройки в VariousHacks.ini.
